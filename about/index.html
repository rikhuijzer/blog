This website

This website is built with Franklin.jl and the Julia programming language. Most posts on this site contain math, plots and the code to reproduce the results. Adding the code makes reading the posts more difficult. However, most statistical calculations cannot be done by hand. For example, fitting a simple linear model requires minimizing the sum of squares. By adding the code, this blog aims to show that statistical results are not created by some magical procedure.

For example, a basic plot can be as simple as

using Gadfly

range = 1:30
p = plot(x = range, y = [x^2 for x in range])

The source code of this website is available on GitHub.
