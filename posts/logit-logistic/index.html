The logit and logistic functions

2020-11-04

Linear regression works on real numbers R\mathbb{R}R, that is, the input and output are in R\mathbb{R}R. For probabilities, this is problematic because the linear regression will happily give a probability of −934-934−934, where we know that probabilities should always lie between 000 and 111. This is only by definition, but it is an useful definition in practice. Informally, the logistic function has been designed to convert values from real numbers to probabilities and the logit function is the inverse.

  1. Logistic
  2. Logit

Logistic

The logistic function converts values from (−∞,∞)(-\infty, \infty)(−∞,∞) to (0,1)(0, 1)(0,1):

logistic(x)=11+e−x. \text{logistic}(x) = \frac{1}{1 + e^{-x}}. logistic(x)=1+e−x1​. 
logistic(x) = 1 / (1 + exp(-x))

We can visualise this with

using AlgebraOfGraphics
using CairoMakie

I = -6:0.1:6
xy = mapping([I] => :x, [logistic.(I)] => :y) * visual(Lines)

draw(xy)

Logit

The inverse of the logistic function is the logit function,

logit(x)=log⁡(x1−x). \text{logit}(x) = \log(\frac{x}{1 - x}). logit(x)=log(1−xx​). 
logit(x) = log(x / (1 - x))

This function goes from (0,1)(0, 1)(0,1) to (−∞,∞)(- \infty, \infty)(−∞,∞).
