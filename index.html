Hi! I'm a PhD student at the University of Groningen. See about for more information about me.

Most posts on this site are about statistics and programming. The posts contain math, plots and the code to reproduce the results.

For example, a simple plot looks as follows:

using AlgebraOfGraphics
using CairoMakie

I = 1:30
xy = mapping([I] => :x, [I.*2] => :y)

draw(xy)

The source code of this website is available on GitHub.
